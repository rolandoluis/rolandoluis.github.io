Pattern Role

Qué representa

Este patrón explora una forma visual de comunicar espera o actividad al usuario sin depender de texto plano.

Preview

Vista previa

Aquí iría la demo o preview embebida del patrón.

Código


.loader {
  width: 64px;
  height: 64px;
  border-radius: 50%;
  border: 6px solid #2a2f3a;
  border-top-color: #6ea8fe;
  animation: spin 1.2s linear infinite;
}

Cuándo usarlo

En estados transitorios donde el sistema aún no tiene un progreso porcentual real.

Assessment

Evaluación

Patrón útil como referencia visual, aunque no necesariamente protagonista dentro del sistema actual.
