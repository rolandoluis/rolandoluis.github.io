Circular Loading Indicator

Patrón visual de feedback para estados de carga con animación circular y ajuste dinámico de velocidad.

Role

Qué representa

Esta pieza explora un patrón visual para comunicar actividad del sistema mientras una interfaz carga o prepara contenido. No busca medir progreso real, sino ofrecer una señal clara de espera.

Contexto

Los estados de carga suelen necesitar una pista visual que reduzca incertidumbre. Este patrón propone una solución simple basada en un círculo con rotación continua.

Cuándo usarlo

En cargas breves o estados de espera sin porcentaje real disponible, donde un indicador de actividad resulta más adecuado que una barra de progreso.

    inset 0 0 18px rgba(110,121,255,0.25);
  animation: spin 6s linear infinite;
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}
        

const ring = document.getElementById("loaderPreviewCircle");
const speed = document.getElementById("loaderSpeed");
const label = document.getElementById("loaderSpeedValue");

function updateLoaderSpeed() {
  label.textContent = speed.value + " %";
  ring.style.animationDuration = (60 / speed.value) + "s";
}

speed.addEventListener("input", updateLoaderSpeed);
updateLoaderSpeed();

Notes

Observaciones

Valor técnico

Sirve como patrón base de feedback visual y como referencia para componentes de carga simples.

Limitaciones

No comunica progreso real. Es un indicador de actividad, no una métrica de avance.

Lectura actual

Más útil como referencia de biblioteca visual que como pieza central del sistema actual.

Evolución posible

Podría derivar hacia loaders más integrados, skeleton states o variantes de feedback contextual.

Assessment

Evaluación

Patrón válido como memoria técnica y referencia visual. Conserva valor dentro del laboratorio como pieza archivada de exploración frontend.
