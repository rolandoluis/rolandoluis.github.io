Variables y tipos

En JS moderno, usa const y let...

const name = "Rolando";
let count = 0;

function inc() {
  count += 1;
  return count;
}
Tip
Prefiere const por defecto; usa let solo cuando reasignes.

DOM y eventos

Seleccionar elementos

const btn = document.querySelector("#saveBtn");
btn.addEventListener("click", () => alert("OK"));
Advertencia
No uses inline handlers (onclick=""). Mantén JS separado.
