Implementación de Estructura de Dominio y Backup Centralizado

Rolando Luis Martín · 8 Mar 2026 · Sistemas · Nivel: Intermedio

Guía técnica estructurada para la implementación de un entorno profesional de dominio y almacenamiento centralizado para 25 usuarios[cite: 225, 226].

Contenido
Figuras
Nota
Esta configuración está diseñada para una empresa con cuatro departamentos clave: Dirección, Recursos Humanos, Producción y Comercialización[cite: 227].

Objetivo

Optimizar la seguridad y disponibilidad de los datos mediante un Controlador de Dominio (DC) y la integración de un NAS de 8TB como repositorio central[cite: 227].

Contexto

La gestión centralizada en Windows Server permite aplicar permisos granulares y facilitar la auditoría de acceso para los 25 usuarios del entorno[cite: 235, 226].

Desarrollo

1. Usuarios y Grupos en Active Directory

Utilizaremos Unidades Organizativas (OU) para segmentar las políticas bajo una unidad principal denominada "Empresa_Usuarios"[cite: 229, 230]:

  • OU Dirección: Asociada al grupo de seguridad G_Direccion[cite: 231].
  • OU Recursos Humanos: Asociada al grupo G_RRHH[cite: 232].
  • OU Producción: Asociada al grupo G_Produccion[cite: 233].
  • OU Comercialización: Asociada al grupo G_Comercial[cite: 234].

2. Diseño del Almacenamiento en el NAS

La ruta raíz \\NAS-BACKUP\Empresa_Data reflejará la jerarquía organizacional para facilitar la sincronización y cuotas de disco[cite: 239, 252]:

\Direccion
  \Backups_Area (Datos operativos)
  \Perfiles (Redirección de carpetas)
\RRHH
  \Backups_Area
  \Perfiles
... (Repetir estructura para Producción y Comercial)
[cite: 240, 241, 242, 243, 244, 245, 246, 249]

3. Implementación de GPO: Redirección de Carpetas

Configuraremos una directiva para que la información de los terminales se guarde automáticamente en el servidor[cite: 254]:

  • Ruta: Configuración de usuario > Directivas > Configuración de Windows > Redirección de carpetas[cite: 257].
  • Destino: \\SERVIDOR\Perfiles\%USERNAME%[cite: 260].
Tip de Seguridad
Desmarca "Otorgar al usuario derechos exclusivos" para permitir que los administradores realicen backups de las carpetas redirigidas[cite: 261].

Estrategia de Backup

Se recomienda una sincronización incremental programada (ej. Synology Active Backup) con la siguiente frecuencia[cite: 264, 265]:

  • Copia Completa: Semanal[cite: 265].
  • Copia Diferencial: Diaria para optimizar los 8TB de capacidad[cite: 265].

Conclusión

La integración de Active Directory con un almacenamiento NAS centralizado no solo simplifica la administración de los 25 usuarios, sino que garantiza la soberanía de los datos. Al utilizar GPOs de redirección de carpetas, se elimina la dependencia del hardware local de los empleados, asegurando que la información crítica de departamentos como RRHH o Dirección esté siempre respaldada y disponible para procesos de recuperación ante desastres.
